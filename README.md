# Student Grade Tracker (Part 1)

This project is a Python-based Student Grade Tracker developed as part of an assignment.

## Features
- Cleans and processes raw student data
- Calculates grades using loops and conditions
- Generates class performance summary
- Performs string manipulation tasks

## File Included
- part1_grade_tracker.ipynb

## How to Run
Open the notebook in Jupyter/Colab and run all cells to see outputs.
